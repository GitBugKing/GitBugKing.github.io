North American Bike Sales Dashboard

In this project, I developed a comprehensive Business Intelligence (BI) dashboard to showcase North American bike sales data. The dashboard integrates data from multiple sources, centralized through a data warehouse, enabling efficient data analysis and reporting.

See Details

Key Technologies:

  • Database: Utilized PostgreSQL hosted on AWS RDS (Relational Database Service) to store and manage bike sales data.
  • Data Warehouse: Built a data warehouse to integrate various data sources, facilitating unified data analysis and reporting.
  • ETL Process: Designed and implemented an ETL (Extract, Transform, Load) process to extract raw data into the data warehouse, perform data cleansing and transformation.
  • Data Visualization: Developed the BI dashboard using Tableau to visually present key sales metrics and trends.
  • Performance Optimization: Optimized database performance, including index optimization, query optimization, and database schema optimization to ensure fast data retrieval and reporting.

Project Highlights:

  • Integration of Multiple Data Sources: The centralized data warehouse integrated data from various sources, providing a unified data view.
  • Efficient Data Visualization: Developed interactive dashboards to display key performance indicators (KPIs), helping business users quickly understand and analyze sales data.
  • Database Performance Optimization: Enhanced database performance through index and schema optimization, significantly improving query performance and response time, supporting large-scale data processing and analysis needs.

This project demonstrates my comprehensive skills in data processing, database management, and data visualization, effectively transforming complex data into valuable business insights.
